Los archivos contenidos en esta carpeta contienen los scripts para la creacion de la BD y los datos basicos en las tablas necesarias.

El archivo "script_inicial" cuenta con el script de la estructura de la BD, los triggers y la insercion de datos del usuario root y las medidas basicas.

El archivo "triggers" tiene los triggers separados.

El archivo "modelado" cuenta con un diagrama entidad relacion realizado en Workbench.
